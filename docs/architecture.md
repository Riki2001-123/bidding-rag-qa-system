# V1 架构说明

## 数据流

1. 用户登录后下载对应业务域的 Excel 模板
2. 上传 Excel 到 `/api/imports/{domain}/excel`
3. 后端完成：
   - 列名标准化
   - 必填字段校验
   - 结构化字段写入数据库
   - 语义字段递归切块
   - 向量化并写入向量索引
4. 检索时先按用户权限过滤
5. 先查结构化字段，再补充语义召回
6. LLM 基于检索结果回答，并返回引用与附件

## 切块策略

- 主数据是 Excel 记录，不对整行记录暴力拆碎
- 只对长文本字段做递归切割
- 默认优先按段落、句号、分号等语义边界切割
- 默认 `max_chars=400`，`overlap=60`

## 权限策略

- `admin`：全部可见
- `internal`：可见 `public/internal` 且必须属于授权项目
- `supplier`：仅可见 `public` 且必须属于授权项目
