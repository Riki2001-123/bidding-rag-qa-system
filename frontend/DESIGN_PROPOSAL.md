# 招投标智能问答系统 — 前端设计方案

## 设计定位

**AI Chat 产品 + B2B 企业级 + 数据密集型**

系统是给招投标从业人员用的专业工具，不是消费品。设计需要传达三种感觉：**专业可信**（企业级数据系统）、**智能高效**（AI 赋能）、**现代精致**（不是粗糙的内部工具）。

---

## 方案选型：Linear + Claude 混合风格

从 54 个顶级设计系统中，我推荐 **Linear 为主、Claude 为辅** 的混合方案。理由：

| 维度 | Linear（主） | Claude（辅） |
|------|-------------|-------------|
| 暗色主题 | 业界天花板，`#08090a` 近黑背景，信息通过白色透明度层级管理 | 暖色调暗色（`#141413`），带橄榄绿温度 |
| 精密感 | 极致工程美学，每个像素都有精确的层级关系 | 编辑式排版，温暖的人文气质 |
| 适合场景 | 数据密集型仪表盘、专业工具、开发者产品 | 对话式产品、内容展示、阅读体验 |
| 配色 | 几乎无彩色，仅品牌靛蓝 `#5e6ad2` + 绿色状态色 | 赤陶色 `#c96442` 暖色调 |
| 字体 | Inter Variable，负 letter-spacing 压缩标题 | Serif + Sans 双字体，杂志式排版 |

**为什么适合你的项目**：招投标系统需要处理结构化数据（标书、企业、政策），Linear 的"精密仪器感"匹配这种专业工具定位。对话部分借鉴 Claude 的"温暖对话感"，让 AI 回复不那么冰冷。

## 核心设计系统

### 1. 色彩体系

```
背景层级（从深到浅，创造深度感）：
  页面底色     #08090a    （Linear Marketing Black，最深背景）
  侧边栏背景   #0f1011    （Linear Panel Dark，面板层）
  卡片/浮层    #191a1b    （Linear Level 3，凸起表面）
  悬停态       #28282c    （Linear Secondary，最浅暗色）

文字层级（通过透明度递减，非色相变化）：
  主文字       #f7f8f8    （近白，带微暖色温）
  次要文字     #d0d6e0    （冷银灰，正文描述）
  辅助文字     #8a8f98    （弱化文字，placeholder/元数据）
  最弱文字     #62666d    （时间戳、禁用态）

品牌色：
  主色调       #5e6ad2    （Linear 靛蓝，用于 CTA、活跃态、品牌标识）
  亮态         #7170ff    （悬停态、选中态）
  发光态       #828fff    （按钮 hover glow）

领域色彩（保持你原有的三色系统，调整饱和度适配暗色）：
  招标         #f59e0b    （琥珀金）
  政策         #3b82f6    （钴蓝）
  企业         #10b981    （翡翠绿）

边框系统（核心！这是 Linear 质感的灵魂）：
  最弱边框     rgba(255,255,255,0.05)    （几乎不可见，但创造结构）
  标准边框     rgba(255,255,255,0.08)    （卡片、输入框）
  强调边框     rgba(255,255,255,0.12)    （分割线、重要区域）
  聚焦边框     #5e6ad2                   （输入框聚焦态）
```

### 2. 字体

```css
/* 标题和UI文字 */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", sans-serif;

/* 大标题用负 letter-spacing 压缩，Linear 标志性做法 */
.hero-title {
  font-size: 32px;
  font-weight: 510;          /* Linear 独特的 510 体重 */
  letter-spacing: -0.7px;    /* 压缩感 */
  line-height: 1.15;
  color: #f7f8f8;
}

/* 正文 */
body-text {
  font-size: 14px;
  font-weight: 400;
  line-height: 1.65;
  color: #d0d6e0;
}

/* 辅助文字 */
helper-text {
  font-size: 12px;
  color: #8a8f98;
}
```

### 3. 阴影与深度

```css
/* 不用传统 border，用 shadow-as-border（Vercel 发明，Linear 推广） */
.card {
  box-shadow: 0px 0px 0px 1px rgba(255,255,255,0.08);
  background: #191a1b;
  border-radius: 12px;
}

/* 悬浮态：加微妙发光 */
.card:hover {
  box-shadow: 0px 0px 0px 1px rgba(255,255,255,0.12),
              0 8px 24px rgba(0,0,0,0.3);
}

/* 输入框聚焦 */
.input:focus-within {
  box-shadow: 0px 0px 0px 1px #5e6ad2,
              0px 0px 0px 3px rgba(94,106,210,0.15);
}
```

## 各页面设计方案

### A. 导航栏（毛玻璃 + 极简）

```
┌──────────────────────────────────────────────────────┐
│ ◆ 招投标智能问答        [智能问答] [数据检索] [系统总览]   👤用户 │
└──────────────────────────────────────────────────────┘
  - backdrop-filter: blur(20px)，半透明深色背景
  - 导航 tab：活跃态底部 16px 宽的靛蓝色下划线，滑入动画
  - 高度 56px（比原来 64px 更紧凑）
  - Logo 文字用 510 体重 + letter-spacing: 0.5px
  - 用户区域：Dropdown 点击展开退出菜单
```

```

┌──────┐               ┌──────────────────┐
│  🤖  │               │ 🤖 智能问答    ▼ │
│      │               │ [+ 新对话]       │
│  📄  │               │ [🔍 搜索对话...]  │
│  📄  │               │                  │
│  📄  │               │ 📄 标书查询测试   │
│  📄  │               │ 📄 政策解读...    ✕│
│      │               │ 📄 企业信息查...  ✕│
│      │               │                  │
│      │               │ 自动识别问题领域   │
```

**用户消息**：

```
│     │ 这家公司最近中标过哪些   │ │  ← 半透明靛蓝卡片背景
│     │ 项目？                   │ │    rgba(94,106,210,0.08)

└─────────────────────────────────┘
```

**AI 消息**：
```
┌──────────────────────────────────────────────┐
│ 🤖  │ 招标                                    │  ← 领域标签
│     ├────────────────────────────────────────┤  ← 左侧 2px 彩色边框
│     │ 根据查询结果，该公司近半年中标了以下项目：│     （颜色随领域变化）
│     │                                        │
│     │ 1. XX智慧校园建设采购项目，中标金额...   │
│     │ 2. XX云计算服务项目，中标金额...        │
│     │                                        │
│     │ 引用来源：[标书A...] [标书B...] [标书C] │  ← Tag 药丸
│     │                                        │
│     │ 📋复制  👍  👎                          │  ← 操作栏（悬停显示）
```

**设计要点**：
- AI 消息左侧 2px 彩色竖线（招标=#f59e0b / 政策=#3b82f6 / 企业=#10b981）
- 操作栏默认隐藏，hover 消息时渐入
- 引用来源用半透明药丸 Tag，不是实心色块
- 打字指示器：三个点脉冲动画（绿色 #5e6ad2 靛蓝色）

## 关键设计原则（从 2026 趋势提取）

1. **冷静设计（Calm Design）**：减少视觉噪音，每个元素都要"挣得"自己的位置。去掉所有不必要的装饰。

2. **战略性极简主义**：不是简单删东西，而是让留下的每个元素都有精确的功能。配色从 5 色减到 3 层（品牌色 + 领域色 + 灰度层级）。

3. **渐进式披露**：操作栏默认隐藏，hover 渐入。侧边栏可折叠。搜索框在有多个会话时才有价值。

4. **情感化设计进入 B2B**：打字动画、过渡动画、微妙的 hover 反馈，让专业工具不冰冷。

5. **shadow-as-border**：不用传统 CSS border，全部用 `box-shadow: 0px 0px 0px 1px` 创造更精致的边线。

6. **信息密度通过透明度管理**：不用不同颜色区分层级，用白色的不同透明度（88% → 82% → 53% → 39%）。

## 与现有方案的对比

| 维度 | 当前方案 | 新方案 |
|------|---------|--------|
| 主色调 | 翡翠绿 #10b981 | 靛蓝 #5e6ad2（Linear 标志色） |
| 背景色 | #0f172a（Tailwind slate-900） | #08090a（更深，近乎纯黑） |
| 文字色 | 白色 + 透明度 | Inter 字体 + 510 体重 + 负 letter-spacing |
| 边框 | 实线 border + 半透明白 | shadow-as-border（更精致） |
| 卡片 | 实心背景色块 | 微弱透明度 + 多层阴影 |
| 侧边栏 | 固定宽度，无折叠 | 可折叠 280px ↔ 56px |
| 操作栏 | 始终显示 | 悬停消息时渐入 |
| 搜索 | 侧边栏内嵌 | 侧边栏内嵌，折叠态隐藏 |
| 动画 | 基础过渡 | 导航下划线滑入 + 光晕脉冲 + hover 渐入 |

## 执行优先级

如果要做，建议分两步：

**第一步（视觉冲击最大）**：
1. 配色体系切换（靛蓝主色 + 更深的背景层级）
2. shadow-as-border 替换所有 border
3. 字体 Inter + 负 letter-spacing 标题
4. 空状态重做（光晕 + 2x2 卡片网格）

**第二步（体验细节）**：
5. 侧边栏折叠功能
6. 消息操作栏 hover 渐入
7. 输入框聚焦发光效果
8. 导航下划线动画
