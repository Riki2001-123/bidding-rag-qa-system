# 浅色高级感设计方案 V2

## 设计方向
**Notion + Claude Light Mode + Apple HIG** 混合风格，完全脱离暗色系。

## 核心原则
1. 暖白留白，背景用 #fafafa 微暖白
2. 做减法，去掉彩色光晕、shadow-as-border、发光效果
3. 呼吸感，消息间距增大，操作栏克制
4. 排版秩序，通过对齐和间距传达品质

## 配色
- 页面背景 #fafafa | 卡片 #ffffff | 侧边栏 #f5f5f5
- 主色 #2563eb | 文字 #111827 / #6b7280 / #9ca3af
- 边框 #e5e7eb | 用户气泡 #2563eb

## 组件
- 头像：AI用白底+SVG图标，用户用蓝底+首字母
- AI消息：无背景纯文字 | 用户消息：右对齐蓝色圆角气泡(16px)
- 输入框：大圆角白色容器
- 动效：极克制，只保留hover过渡
